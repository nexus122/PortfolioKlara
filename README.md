## Instalación
### Dependencias:
- Node.js
- npm

## Pasos:
### Clonar el repositorio:
```
git clone https://github.com/nexus122/PortfolioKlara.git
```
### Instalar las dependencias:
```
npm install
```
### Iniciar el servidor de desarrollo:
```
npm start
```
